# PeqSystem Refatorado

Sistema de vendas desenvolvido em **Java**, refatorado aplicando princípios **SOLID** e arquitetura **MVC**.

## Funcionalidades
- Cadastro de produtos
- Registro de vendas
- Resumo das vendas
- Finalização de vendas

## Tecnologias
- Java
- Swing (Interface gráfica)
- Git e GitHub
